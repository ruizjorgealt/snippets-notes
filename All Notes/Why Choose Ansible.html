Extensions

These extensions provide additional custom functionality.

Extensions comes into three categories:
  1. Modules
    1. Many provided
    2. Most common extension
    3. Found in expected paths
      1. /usr/share/ansible/sitemod.py​
  2. Plugins
    1. Extends core
    2. Logging/display callbacks
    3. Communication
    4. Template filters 
  3. Dynamic inventories
    1. Executable
    2. Returns JSON data
    3. Infra as a service (IaaS)
    4. Existing IT asset management

Ansible Advantages

Playbooks Tasks and Modules
  • Enough Abstraction
    • Desired state in simple text, not code
    • Easy-to-view changes
  • Top-Down Execution
    • Order – Can be expected and followed
    • Easy-to-express dependencies
  • Open source
  • Scalability
  • Quick learn
  • Recuded toolbox
    • Ad Hoc

      • Complicated application upgrades
