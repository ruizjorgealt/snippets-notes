C-strings

C-Strings

C-strings in C++ are a data type that one could consider a bit more complex and flexible than strings. 
This is a data type that consists of an array of characters terminated by a null character at the end.

Declaring a c-string

To declare a c-string variable
​char var;
​
​Assigning a character value to a c-string
​var = ‘A’;

​Declaring a c-string variable and pre-determining the size
//“n” DETERMINES THE NUMBER OF CHARACTERS THIS C-STRING CAN HOLD
​char var[n];

​​Assingning a value to a c-string with value greater than 1
​char var[n] = “Jorge”;
​

Understanding c-strings

C-strins tend to be more complex due to our ability of individually being able to work with them. 
For example, after we perform the following assignment:
​char var[5] = “jorge”;

We can access each value on the c-string individually by placing a bracket and a value in front of the c-string variable
​cout << var[0];
​The previous statement would simply output the letter ‘j’. This is beause we are specifically demanding the array of characters to provide us with the character value placed at the 0 place within the array.

Structure of c-strings

Our c-strings are nothing but arrays of chracters that we start at the 0 place/slot.
For example the following assignment of a c-string could visually explained in the following manner.

​//DECLARING C-STRING VARIABLE TYPE
char var[5] = “jorge”;
​
[ j ][ o ][ r ][ g ][ e ][ \0 ]
  |     |     |     |      |      |
  0   1     2    3     4     5

Another way of visualizating a c-string
var[0] —> j
var[1] —> o
var[2] —> r
var[3] —> g
var[4] —> e
var[5] —> Null

To copy a c-string

To copy a c-string into another c-string, we must first make sure that the new c-string is also declared as an array and not just a single chracter. Remember, to declare an array of chracters we do it in the following manner:
​char var[n];​ 
Remember that we will be erasing the contents of the variables where we will be copying our new array of characters.
Follow this model: ​strcpy(target,source)​
Both must be c-strings!
#include <iostream>
using namespace std;
int main(int argc, char *argv[]) {
//DECLARING TWO CHAR VARIABLES OF THE
//SAME SIZE
char var1[10], var2[10];
//READ INTO VARIABLE
cin >> var1;
//PERFORM COPY
strcpy(var2, var1);
//DISPLAY
cout << endl << var2;
return 0;
}

To add a c-string

This function allows us to add two c-strings. The contents of the target variable will be modified.
Follow this model: ​strcat(target,source)​
#include <iostream>
using namespace std;
int main(int argc, char *argv[]) {
//DECLARING CHAR VARIABLES WITH DIFFERENT VALUES
char var1[10] = "Jorge ";
char var2[10] = "Ruiz";
//PERFORMS ADDITTION
strcat(var1, var2);
//DISPLAY
cout << var1;
return 0;
}

Comparing c-strings

The following function allows us to compare c-strings in order to be used with an if statement.
To perform the comparison we use the function ​strcmp(var1, var2);​.
#include <iostream>
using namespace std;
int main(int argc, char *argv[]) {
//DECLARING CHAR VARIABLES WITH THE SAME VALUES
char var1[10] = "Hello";
char var2[10] = "Hello";
//COMPARING BOTH C-STRINGS WITHING AN IF STATEMENT
if(strcmp(var1, var2)==0){
cout << "Yes, both c-strings are equal and they contain the word: ";
}
//DISPLAY
cout << var1;
return 0;
}

Converting character case

​​The following functions allow for char variables to be modified.

Converts the chracter to uppercase.
​toupper(var);​

Converts the character to lowercase.
​tolower(var);

Checks if character is lowercase.
​islower(var);​

Checks if character is uppercase.
​isupper(var);​

Checks if character is alphabetic.
​isalpha(var);​

#include <iostream>
using namespace std;
int main(int argc, char *argv[]) {
//DECLARING CHAR VARIABLE
char var;
//INPUT DATA INTO CHAR
cin >> var;
//CONVERT CHARACTER INTO UPPERCASE
var = toupper(var);
//DISPLAY THE UPPER CASE
cout << var;
return 0;
}
Notice that when trying to convert an entire c-string into either lowercase or uppercase, we would need to convert every single character using the properties of a char array.

​var[5] = toupper(var[5]);

Length of c-string

​strlen()​ gets the length of the char array, or the total number of elements used.
This is a function that requires the array to be sent as an argument.
​int var = strlen(chArray);​
#include <iostream>
using namespace std;
int main(int argc, char *argv[]) {
char c[100];
cout << "Enter a sentence: ";
cin >> c;
cout << "Your sentence is about " << strlen(c) << " characters long.";
return 0;
}

Copying and appending c-strings

​strncopy​ copies the c-string pointed by the source into the array pointed by the destination, including the termination null characters.

​strncopy(str01,str02);​ ←Copies strings str02 into str01.
​strncopy(str01,str02,3);​ ←The first 3 characters of str02 will be copied to str01.
​strncopy(str01,”cool word”);​ ←We can also copy string literals into strings.

​strncat​ appends one string to the other. The syntax is the same as ​strncopy​.
