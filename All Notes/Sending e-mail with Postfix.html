Sending e-mail with Postfix

Install the following packages
  • netcal
  • libsasl2-modules
  • postfix
  • mailutils
  • ssmtp
apt-get install netcat libsasl2-modules postfix mailutils ssmtp
While installing Postfix select Internet Site for configuration.

If you wish to modif the fully qualified domain name for your server edit the Postfix config file.
sudo nano /etc/postfix/main.cf
Edit the following line:
myhostname = fqdn.example.com
Also edit:
inet_interfaces = localhost
Then reload Postfix
service postfix reload
service postfix restart
To reconfigure:
dpkg-reconfigure postfix

Sending an email

There are multiple ways of sending email using netcat.
Sending a simple e-mail through the prompt:
echo "Body of e-mail." | mail -s 'Subject' jorgearuiz@icloud.com
Sending contents of a file:
mail -s "Subject" jorgearuiz@icloud.com < file.txt

Troubleshooting

Look into the following log to inspect for errors
/var/log/mail.log
