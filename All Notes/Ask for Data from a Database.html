Ask for Data from a Database

SELECT

Tells the database we want some information returned to use.
See all the first names in the database:
SELECT first_name
FROM people;
Sell all the last names first, then the first names
SELECT last_name, first_name
FROM people;

WHERE

Adds condition to a statement
Selecting all of the people that live in California:
SELECT *
FROM people
WHERE state="CA";
Selecting people who asked for a shirt or hat
SELECT first_name, last_name, shirt_or_hat
FROM people
WHERE shirt_or_hat="hat";

To add more criteria to a select statement, we can add multiple conditions:
SELECT first_name, last_name
FROM people
WHERE state='CA' AND shirt_or_hat='hat';

The IS and AND operator can be used interchangeably.
SELECT first_name, last_name
FROM people
WHERE state='CA' AND shirt_or_hat IS 'hat';
If you want to filter a value that is not what you are looking for, then use IS NOT or !=
SELECT first_name, last_name
FROM people
WHERE state != 'CA' AND shirt_or_hat IS NOT 'hat';

Grouping Conditions

It is important to correctly group conditions in order to fetch the proper data we are requesting.
In the following example, we used parentheis () to group our conditions.
SELECT first_name, last_name, shirt_or_hat, team
FROM people
WHERE state != 'CA' AND shirt_or_hat IS NOT 'shirt' AND (team IS "blue" OR team IS "green");

LIMIT Responses

To limit the number of responses you obtain when running your queries, you can simply use the LIMIT operator.
SELECT first_name
FROM people
WHERE first_name LIKE 'A%'
LIMIT 5;
"Anne"
"Ann"
"Annie"
"Anthony"
"Angela"
To skip 3 rows and get five names starting from that row:
SELECT first_name
FROM people
WHERE first_name LIKE 'A%'
LIMIT 5 OFFSET 3;
"Anthony"
"Angela"
"Antonio"
"Albert"
"Anna"

ORDER BY

Sets the sort order of returned records based on a given criteria.

The following query will order the first name, last name, by state. 
States will be ordered alphabetically.
SELECT first_name, last_name, state
FROM people
ORDER BY state ASC;
ASC – From small to large.
DESC – From large to small.
Combining two ORDER BY operators
SELECT state, first_name, last_name
FROM people
ORDER BY state DESC, first_name ASC
LIMIT 10;

Find information about the data

LENGTH( ) → Number of chars in the string
DISTINCT( ) → Yields unique values
COUNT( ) → Counts the instances of a value
SELECT DISTINCT(last_name)
FROM people
LIMIT 5;
"Howell"
"Alvarez"
"Olson"
"Garcia"
"Rice"
Counting how many people are from California
SELECT COUNT(*)
FROM people
WHERE state='CA';
