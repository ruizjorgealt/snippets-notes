LDAP Command-Line Tools

Configure encrypted password for our LDAP server. This is highly important becaus some LDAP commands expect encrypted passwords in configuration.
How to configure an encrypted password:
The step is to generate an encrypted password
slappasswd
You should now have something like this:
{SSHA}00Zpnx28VCenx4ZjSDYhNk9M+3IVuK/3
That password can be used in LDIF statements such as:
dn: olcDatabase={0}config,cn=config
changetype: modify
replace: olcRootPW
olcRootPW: {SSHA}00Zpnx28VCenx4ZjSDYhNk9M+3IVuK/3
dn: olcDatabase={1}hdb,cn=config
changetype: modify
replace: olcRootPW
olcRootPW: {SSHA}00Zpnx28VCenx4ZjSDYhNk9M+3IVuK/3
-
replace: olcRootDN
olcRootDN: cn=admin,dc=ec2-34-209-24-204,dc=us-west-2,dc=compute,dc=amazonaws,dc=com
-
replace: olcSuffix
olcSuffix: dc=ec2-34-209-24-204,dc=us-west-2,dc=compute,dc=amazonaws,dc=com
The LDIF statement can be then executed through the command line
ldapmodify -a -Q -Y EXTERNAL -H ldapi:/// -f pass.ldif
-a ← To make a new entry
-Q ← Quiet mode and avoid unwanted prompts
-Y EXTERNAL ← Enable the use of low level protocol like IPC
-f ← Getting input from a file

LDAP Search

  • Enables you to search for and retrieve specific entries in the directory
  • Two mandatory arguments are required for this command
Mandatory Arguments Descriptions
-b "basedn" Specifies the base DN for the search
-s scope Specifies search scope: base, one, or sub
LDAP search command:
ldapsearch -p 389 -h myhost -x -z 2 -b "your base DN" -s base -v "objectclass=*"
-p ← Port number of LDAP server
-h ← Hostname
-z ← Maximum number of results to return
-v ← Search filter
-x ← For simple authentication
Command to search for an entity
You do not need to pass port not a host if you are using it on your localhost
sudo ldapsearch -x -z 200 -b 'dc=example,dc=com' -s sub -v '(uid=davis)'
Command to search for entities with OR operator
sudo ldapsearch -x -z 200 -b 'dc=example,dc=com' -s sub -v '(|(uid=sara) (uid=davis))'
Command to add entities using ldapadd
  • Used to add entries, their object classes, attributes, and values to the directory
  • One mandatory argument required for this command
Mandatory Arguments Description
-f "filename" Specifies the input filename
ldapadd -x -W -D 'cn=admin,dc=hostname,dc=com' -f ldif_file_name.ldif
-p ← Port number of LDAP server
-h ← Hostname
-D ← Bind ID to bind with LDAP server
-W ← To enable password prompt
-x
command to delete entities using ldapdelete
ldapdelete -x -W -D "cn=admin,dc=hostname,dc=com" "uid=davis,ou=people,o=yyy,c=uk,dc=hostname,dc=com"
