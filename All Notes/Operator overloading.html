Operator overloading

Operator Overloading

Operator overloading allow us to perform basic arithmetic and assignment functions with objects. 
  • One of the main reasons for overloading the assignment operator with objects is to avoid a shallow copy.
#include <iostream>
#include "Sally.h"
using namespace std;
int main(int argc, char *argv[]) {
Sally s1(90);
Sally s2(10);
Sally s3;
//OPERATOR OVERLOADING IN ACTION
s3 = s1 + s2;
cout << s3.num;
return 0;
}
//
// Sally.h
// Created by Jorge A. Ruiz on 12/31/17.
#ifndef _Sally_h_
#define _Sally_h_
class Sally {
public:
int num;
Sally();
Sally(int);
//OPERATOR OVERLOADING FUNCTION OF THE SALLY RETURN TYPE
Sally operator+(Sally);
virtual ~Sally();
};
#endif
//
// Sally.cpp
// Created by Jorge A. Ruiz on 12/31/17.
#include "Sally.h"
//RETURN TYPE IS SAME AS CLASS NAME
//OPERATOR OVERLOADING FUNCTION
Sally Sally::operator+(Sally s){
Sally brandNewObj;
brandNewObj.num = num + s.num;
return (brandNewObj);
}
Sally::Sally(int a) {
num = a;
}
Sally::Sally() {
num = 0;
}
Sally::~Sally() {
}
operator is a keyword indicating that an operator is being overloaded. The operator keyword is followed by the operator being overloaded, the assignment operator in this case.

Insertion and extraction operators

In order to overload the insertion ​>>​ and extraction ​<<​ operators, we must use friend functions.

Prototype examples:

​friend istream& operator>>(istream&, Person&);

friend ostream& operator<<(ostream&, Person&);​

The actual implementation would look something like this:
​istream& operator>>(istream& strm, Person& p){…}
#include <iostream>
#include "Person.h"
using namespace std;
int main(int argc, char *argv[]) {
Person p;
cout << "Enter name and age: ";
cin >> p;
cout << p;
return 0;
}
//
// Person.h
// Created by Jorge A. Ruiz on 1/1/18.
#ifndef _Person_h_
#define _Person_h_
//#include <iostream>
//#include <string>
using namespace std;
class Person {
private:
string name;
int age;
public:
Person();
friend ostream &operator<<(ostream &output, Person &p);
friend istream &operator>>(istream &input, Person &p);
};
#endif
//
// Person.cpp
// Created by Jorge A. Ruiz on 1/1/18.
#include <iostream>
#include "Person.h"
using namespace std;
Person::Person(){
name = "noname";
age = 0;
}
ostream &operator<<(ostream &output, Person &p){
output << "What the hack " << endl;
output << "my name is " << p.name << " and my age is " << p.age << endl;
return output;
}
istream &operator>>(istream &input, Person &p){
input >> p.name >> p.age;
return input;
}
