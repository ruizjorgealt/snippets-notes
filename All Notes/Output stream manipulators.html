Output Stream Manipulator

In order for one to use the out stream manipulators, we have to make sure and include the following header:
​#include <iomanip>
​

Determininig the number of decimals to display

If one were to limit the number of digits to display when writing a program in C++, make sure to include the following function with the N number of digits that you wish to display.

Notice

This will only works when using decimal numbers or float variables.

For example:
​cout << fixed << setprecission(3) << 458.34586;
Would display the following output:
​458.346
​It automatically rounds the entire decimal to the N number of decimal points that you wish to display.
#include <iostream>
#include <iomanip>
using namespace std;
int main(){
//REMEMBER TO INCLUDE FIXED BEFORE THE SETPRECISSION FUNCTION
cout << fixed << setprecision(1) << 4454.3445;
return 0;
}

Other stream manipulators

Function Performance Example
setw(n) Set the number of space before value of displayed cout << setw(3) << var;
left Displays values on the left section of the prompt cout << left << var;
right Displays values on the right section of the prompt cout << right << right;
setprecision(n) Display a fixed number of decimals cout << fixed << setprecision(3) << var;
