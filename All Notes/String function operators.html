String function operators

String Function Operators

The following functions allow us to perform several operations that provide with information regarding the string.

Length function

Tells you how many characters there are in a string.

To peform and obtain the number of characters in a string use the following method:
​//DECLARE STRING VARIABLE
//ASSIGN VALUE TO STRING VARIABLE
​string state = “California”;
//STORE VALUE DETERMINED BY THE NUMBER OF CHARACTERS IN THE STRING VARIABLE
int size = state.length();

​​Simply add the .length(); extension/function to the string variable to access the number value that corresponds to the length of your string variable.
#include <iostream>
using namespace std;
int main(){
string var;
cin >> var;
//THIS STATEMENT WOULD DISPLAY THE VALUE DIRECTLY
cout << "Your word has " << var.length() << " characters." << endl;
return 0;
}
This program should perform in the following manner
jorge
Your word has 5 characters.

Reading sentences, including space

To get an entire sentence into a string object, including spaces; use the following function.
​getline(cin,var);​
Where var has been declared as:
​string var;

Accessing individual characters

In order to access individual characters from a string variable we would need to access them using a subscript, just as if they were an array.
​string var = “Jorge”;
cout << var[0];​
By using ​var[0]​ we would be accessing the element at the index of 0 which is ‘J’.
