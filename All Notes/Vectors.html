Vectors

Vectors

The standard template library(STL) offers a vector data type, which in many ways is superior to standard arrays.
 
The standard template library(STL) is a collection of algorithms and data types. These are programmer defined, and they are also part of using namespace std(STD).

  • A vector can hold a sequence of values/ elements.
  • A vector stores its elements in continuous memory locations.
  • We can use subscripts to access individual elements in a vector → ​vectorVar[n]​
  • No declarator is needed.
  • Vectors can increase size in order to accomodate new values if vector is already full.
  • Vectors can report the number of elements they contain.

Defining a vector

In order to declare a vector, include the following header.
​#include <vector>​
Now, declare a vector by following the next example.
​vector<int> numbers;​
Where ​<int>​ is the data type and ​number​​s​ is the name of the vector variable.
If you wish to declare a vector and declare the size at the same time, then you achieve that by placing ​()​ containing the size in front of the vector variable.
For example:
​vector<int> numbers(5);​
The 5 is the number of elements that the vector should be able to hold; however, the vector could keep increasing if more elements are needed.
If you wish to initialize all of the 5 elements of the ​numbers​ vector variable to 0, then perform the following changes when declaring the vector.
​vector<int> numbers(5,2);​
Notice that the second argument, the 2 is the values that will be stored in all 5 elements of the vector.
Initializing a vector with the values of another vector.
​vector<int> var02(var01);​
​var02​ will have the same elements and values as ​var01​.

Storing and retrieving values in a vector

We can manipulate vectors as if they were normal arrays by placing ​[n]​, thus allowing us to access the elements through the subscript.

Using the pushback member function

We use ​push_back​ to store a value in a vector that does not have a starting size, or that is already full. This function stores the value in a new element placed at the end of the vector. It “pushes” the value at the end of the vector.
For example:
​var.push_back(25);​
This pushes the 25 to the end of ​var​, thus adding a new element to the vector.

This is quite useful when you need unlimited elements. Or when the number of elements must be specified by the user.
Notice that we would still be able to access by placing the brackets → ​var[i]​

Determining the size of a vector

To obtain the size of a vector, simply append ​.size()​ to a vector variable. This will yield an integer value.
For example:
​vectorSize = vectorVar.size();​

This would be very useful when combined with a for loop.
​for(int i=0; i<vectorVar.size(); i++){
  cout << vectorVar[i] << endl;
}

Passing a vector to a function

The prototype:
​void functionName(vector<int>);
The header:
​void functionName(vector<int> var){...}
Calling the function:
​functionName(var);​

Removing elements from a vector

To remove an element we use the ​pop_back​ member function. The ​pop_back​ member function removes the last element from the vector variable.
​var.pop_back();​

Clearing a vector

Appending the ​.clear()​ member function will clear a vector.
​var.clear();​

Summary of vector member functions

Member Function Description
.at(position) Returns the value of the element located at a particular position in the vector.
.capacity() Returns the maximum number of elements that may be stored in the vector.
.empty Returns true if vector is empty.
.push_back(value) Stores a value in the last element of the vector. If full, new element is created.
.reverse() Reverses the order of the elements in the vector (last becomes first and so on).
.resize(n) Resizes a vector to have n elements, where is greater than the vector's current size.
.resize(n, value) Resizes a vector to have n elements, and populates them with the given value.
.swap(vector2) Swaps the content of the vector with the contents of vector 2.
