Can login from any computer through a browser Requires client to be installed for login
Web based GUI Rich GUI
To install phpLDAPadmin:
sudo apt-get install phpldapadmin
Edit the configuration file to set your host file:
servers->setValue('server','host','jorge.sandbox.csun.edu');
or set to the DNS of your EC2 instance
$servers->setValue('server','host','ec2-18-223-242-216.us-east-2.compute.amazonaws.com');
Set the base of your server:
$servers->setValue('server','base',array('dc=jorge,dc=sandbox,dc=csun,dc=edu'));
Now configure the LDAP bind ID
$servers->setValue('login','bind_id','cn=admin,dc=jorge,dc=sandbox,dc=csun,dc=edu');
For demonstration purposes, you might want to disable template warning:
$config->custom->appearance['hide_template_warning'] = true;
You should now be able to login using www.mysite.com/phpldapadmin
