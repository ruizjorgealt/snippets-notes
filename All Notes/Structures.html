Structures

Structures

C++ allows a set of variables to be combined together into a single unit called a structure. A structure is a programmer-defined data type that can hold many different data values. Although there are not often used in modern programming, it is essential to learn how to use them.
Why? There might be cases where using structures is the solution.

The keyword ​struct​ is used instead of the keyword ​class​ in order to construct another type of abstract data type (ADT).
Structures contain member functions; however, it is recommended to only store variables. Normally they don’t include access specifiers, they default to public (which is the best choice).

For example:
​struct PayRoll{
  int empNumber;
  string name;
  double hours,
         payrate,
         grosspay;
};​

Declaring a structure variable:
​PayRoll var;​

Remember that each instance of a structure will have its own memory allocated to hold its member’s data, therefore, every instance’s members will contain unique data.

Accessing structure members

They are accessed just like public members of a class.
​struct PayRoll{
  int empNumber;
  string name;
  double hours,
         payrate,
         grosspay;
};​

​PayRoll var;

var.name = “Jorge”;​
They function just like any other variable:
​getline(cin, var.name);​
We can apply arithmetic operations:
​int var03 = var01.num * var02.num;​
We can also compare them:
​if(var01.name == var02.name){…}​

Initializing a structure

We can initialize the structure variables with an initialization list or with a constructor.
Writing the structure:
​struct Date{
  int day,
      month,
      year;
}; 
Initializing using an initialization list:
​Date var = {23,5,3};​
The list will automatically assign each of the values to the member variables in the order specified.
For example, the member variables of the instance will contain the following values:
​var.name​     → 23
​var.month​   → 5
​var.year​     → 2

We can also initialize some of the variables if we only wanted to initialize the first two variables, we would have to do the following:
​Date var = {23,5};​
However, notice that it might not work on some compilers.
We also can’t skip the initialization of some variables, for example → ​Date var = {23, ,3};​

Another method of initializing is by using a constructor:
​struct Employee{
  string name;
  int vacationDays,
      daysUsed;
  Employee(string n = “ “, int d = 0){
    name = n;
    vacationDays = d;
    daysUsed = d;
  }
};
//EXAMPLE USING DEFAULT CONSTRUCTOR ON STRUCTURE DATA TYPE
#include <iostream>

daysUsed;
//CONSTRUCTOR
Employee(string n = "NONE", int d = 0){
name = n;

Nested structures

C++ allows the programmer to declare an instance of a structure as a member of another structure.

​struct Costs{
  double wholesale;
  double retail;
};

struct Item{
  string partNum;
  string description;
  Costs pricing;
};​

In order to access the nested structures we do the following:
​Item var;
​Use the ​.​ dot operator​
var.partNum = “123A”;
var.description = “Iron Tool”;
var.pricing.wholesale = 100.00;​   ← Accessing a nested structure’s member variables
var.pricing.retail = 150.00;​         ← Accessing a nested structure’s member variables

Passing structures to functions

They can be passed by value, by reference, and by const reference.
The prototypes:
​void functionName(Item var);                 ← Passing by value
void functionName(Item &var);               ← Passing by reference
void functionName(const Item &var);   ← Passing by constant reference

Returning a structure from a function

Structure variable can also be returned from a function.
#include <iostream>
using namespace std;
struct InvItem{
int partNum;
int quantity;
double price;
string description;
};
InvItem getData();
void displayData(InvItem);
int main(int argc, char *argv[]) {
InvItem Inv;
//THE VARIABLE THAT WILL RECEIVE THE STRUCTURE
//RETURNING FUNCTION TYPE SHOULD BE AN OBJECT OF
//THE STRUCTURE
Inv = getData();
displayData(Inv);
return 0;
}
//STRUCTURE/OBJECT RETURNING FUNCTION
InvItem getData(){
InvItem var;
cout << "Enter the part number: ";

cin.ignore();
cout << "Enter the part description: ";
getline(cin, var.description);
cout << "Enter the quantity of hand: ";
cin >> var.quantity;
cout << "Enter the unit price: ";

}
//PASSING STRUCTURE INSTANCE BY VALUE
void displayData(InvItem i){
cout << endl << "SUMMARY" << endl;

cout << "Part description: " << i.description << endl;
cout << "Unit price: $" << i.price << endl;
}
Now notice that ​InvItem​ is the object return type which is the name of the structure.
All of the elements from one object will be copied into the receiving variable, which is an instance of the structure.

Arrays of structures

One can declare an array of structures by doing it in the same manner in which we declare other types of arrays.
​StructureName var[n];​
To access the members of an array of structures, simply modify the subscript of the array.
​var[n].number​
One could also have arrays within a structure:
​var[n].number[i]​
2D arrays are also a posibility:
​var[n][i].number[y]​
//THE FOLLOWING PROGRAM HAS AN ARRAY OF INTEGERS
//AS A STRUCTURE MEMBER VARIABLE
#include <iostream>
