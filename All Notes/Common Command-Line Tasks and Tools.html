Common Command-Line Tasks and Tools

Pipes ( | ) are used to take the output of one command and send it to another. 

echo – Prints out a string

Piping allows us to use the output of a command as the input of another command. In the following example we will use the command wc which is used to word counting.
echo “hello” | wc
The output of 1 1 6 tells us that there is one word, one line, and six characters.

Peek at files with cat, head, tail, and less

cat – Concatenate and print files
head – See the first ten lines of the file
tail – See the last ten lines of the file
less – Takes a peak at files

The -n argument states how many lines we wish the command to pass to us.
head -n 5 poems.txt​

The following command numbers the lines of text and displays the desired number of lines:
cat poems.txt | cat -n | tail -n 5

Search for text in files and treams with grep

grep – Search files for text that matches a given pattern

Grep is case sensitive

Looks for words containing “the"
grep -n “the” poems.txt​

Edit text with Vim

i and o are for insertion mode.

Vim has two basic modes:
  1. insert mode – Write text as if in normal text editor
  2. normal mode – Provides efficient ways to navigate and manipulate the text
To change between the modes, use esc for normal mode and i for insert mode.

Use the keys h, j, k, and l instead of arrow keys to move the cursor.

h – Left
l – Right
k – Up
j – Down

To navigate the text in terms of words: w, b, and e
w – Moves to the start of next word
e – Moves to the end of the word
b – Moves to beginning of the word

To insert text repeatedly:
30i- Esc ←[NUMBER][i - INSERT][ESC] Inserts - (dash) 30 times after pressing Esc

To find a character:
To find and move to the next (or previous) occurrence of a character, use f and F
For example:
fo ← Finds the next o

3fq ← To find the 3rd occurence of the letter q

Go to matching parenthesis:
In text that is structured with parentheses or brackets, ( or { or [, use % to jump to the matching parenthesis or bracket.

Go to start/end of line
To reach the beginning of a line, press 0
To reach the end of a line, press $
​
Find word under cursor
Find the next occurrence of the word under cursor with *, and find the previous with #​
​
Goto line, g and G
gg​ takes you to the beginning of the file
G takes you to the end of the file
2G takes you to the beginning of second line

Search, /text with n and N
In Vim, you press /, and give the text you are looking for.

You can repeat the search for next and previous occurrences with n and N, respectively.

Insert new line, o and O
To insert text into a new line, press o or O
After new line is created, the editor is set to insert mode.

Removing a character, x and X
x and X delete the character under the cursor and to the left of the cursror, respectively

Replacing the letter under cursor, r
When you need to replace only one character under your cursor, without changing to insert mode, use r.

Deleting, d
d is the delete command

You can combine it with movement, e.g. dw deletes the first word on the right side of the cursor.

It also copies the content, so that you can paste it with p to another location.

d2e ← Removes two words

To repeat the previous command, just press .

Visual mode, v
Besides insert and normal mode, Vim has also visual mode.

In visual mode, you select text using movement keys before you decide what to do with it.

Let's see how. Goto visual mode with v. Then select a word with e. After you've selected the text, you can delete it with d.

Important Vim commands
Most important commands to remember are :w (save), :q (quit), and :q! (quit without saving).

Press u for undo and ctrl+R for redo

If you have a problem, or want to learn more about what Vim offers, type :help

Working with tar archives

tar – Tape archive files

A way of packing various files into one single file.

tar -cvf myfile.tar ~/Desktop/files
Creates an archives based on a directory.

-a ← Tells the type of compression to make based on the file extension
tar -caf myfiles.tar.gz ~/Desktop/files​

Types of compression:
.gz
.bz2

tar -xf myfiles.tar.bz2
Extracts files within the same folder

tar -xf myfiles.tar.bz2 -C ~/Desktop/unpackedfiles/
Extract in a particular destination
