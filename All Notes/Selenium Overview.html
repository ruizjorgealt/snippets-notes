Selenium Overview

Overview of WebDriver



Notice that the RemoteWebDriver class inheris from the FirefoxDriver class.
public class FirefoxDriver : RemoteWebDriver
{
// advanced ctors & Firefox specific drivers
}
The RemoteWebDriver class that the FirefoxDriver inheritcs implements quite a few interfaces.
public class RemoteWebDriver : IWebDriver, IDisposable, ...
{
// implementation of many interfaces
}
The IWebDriver interface defines a number of properties to get or set the browser URL, and also do things.
public interface IWebDriver : ISearchContext, ...
{
// properties to get/set URL, get title, etc.
// methods to perform navigation, close browser, etc.
}
This ISearchContext interface that provides the methods whereby we can find HTML elements on the page.
public interface ISearchContext
{
IWebElement FindElement(By by);
ReadOnlyCollection<IWebElement> FindElements(By by);
}
public class By
{
// static methods to locate elements
}

HTML Element Selection

We can select elements by
  • Id
  • Name
  • ClassName
  • XPath
  • LinkText
  • PartialLinkText
  • TagName
  • CssSelector
When possible choose the Id.

Getting Started in Visual Studio

  • Create DemoWebApp.Tests class library project
  • Install:
    • unit testing framework (xUnit.net)
    • xunit xunit.runner.visualstudio NuGet
    • Selenium.WebDriver NuGet package
using Xunit;
using OpenQA.Selenium;
using OpenQA.Selenium.Firefox;
namespace DemoWebApp.Tests
{
public class LoanApplicationTests
{
[Fact]
pulic void StartApplication()
{
IWebDriver driver = new FirefoxDriver();
drive.Manage().Window.Maximize();
driver.Navigate().GoToUrl("http://localhost:40066/");
IWebDriver applicationButton = driver.FindElement(By.Is("startApplication"));
applicationButton.Click();
/* notice that the driver is synchronized with the firefox web application */
/* the title field in the driver instance constains the title of the current */
/* web page */
Assert.Equal("Start Loan Application - Loan Application", driver.Title)
drive.Dispose();
}
