SSH Basics

The environment 

  • Web Host with SSH access
  • Existing Server
  • Local or Hosted Virtual Machine

What is SSH?

  • Secure SHell
  • Command-line connection to another computer
  • Widely used for system administration of Linux & Mac machines

SSH Clients

  • Mac OS X & Linux: built-in, available in Terminal
  • Windows: PuTTY, others
  • Android: JuiceSSH, others
  • iOS: Prompt, others

SHH Server Considerations

  • SSH Server is a service, not available immediately on boot
  • You can make chamges that prevent access
    • You can cut off your own access
  • SSH depends on a functional network at both ends

Authentication

  • Username & password
    • Specified on the server
  • Key pair
    • Generated by user
    • Public key shared to server
    • Usually more secure

How SSH Works

  • Encrypts the connection between clients and server
  • Usually runs on TCP port 22

Common Messages

Broken pipe: IP changed or server can’t guarantee security

Connecting to an SSH server from Windows

A popular option for the Windows operating system is PuTTY.

 If you need to make a change to the host file on Windows you can edit it in C:\Windows\System32\Drivers\etc\hosts.

Connecting to an SSH server from Mac OS X or Linux

You need to type the ssh command followed by the server name.
If no username is typed, the terminal assumes you want the use the previous username.

ssh scott@orion.dev
Another scenario is that if you have your server running on lets say port 2222, you would connect in the following manner:
ssh -p2222 scott@orion.dev
nano .ssh/known_hosts ← This .ssh folder is inside my home folder. And here, you can see the name of my server and the key information. SSH uses this file to check the host key when you connect to a server and if the server key changes for any reason you'll be notified.

To get this working on Ubuntu

Please remember to install OpenSSH before trying to open a new session your Ubuntu virtual box. It is also important to enable to the bridge adapter on your virtual machine settings.

It is important to install OpenSSH
sudo apt-get install openssh-server
