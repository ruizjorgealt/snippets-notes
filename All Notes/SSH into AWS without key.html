SSH into AWS without key

Follow the instructions in order to SSH into either a Debian or Red Hat based Linux distribution.
1. Log into your ec2 instance using your key
ssh -i "my_key.pem" ubuntu@ec2-3-16-63-249.us-east-2.compute.amazonaws.com
2. Edit the ssh configuration file
sudo vi /etc/ssh/sshd_config
Edit PasswordAuthentication no to PasswordAuthentication yes
3. Give the user a password
passwd ubuntu
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully
At this point you should have the ability to ssh into your your ec2 instance
ssh ubuntu@ec2-3-16-63-249.us-east-2.compute.amazonaws.com
Alternatively, you can create a new user and add it to the sudo group.
sudo adduser <username>
sudo usermod -aG sudo <username>
Do not forget to restart SSHD
sudo systemctl restart sshd
