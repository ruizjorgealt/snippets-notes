Random number generator

Random Number Generator

To generate a random number within a C++ program we need to include the ctime header file.
​#include <ctime>
​This header will allow us to access a value that constantly changes every second.

Random number generator engine

The following function allows us to generate a random value that we will use to generate a random number based on that particular value.

In order to create that random value we will need the help of the ​<ctime>​ header and ​srand()​ function.

To assemble the random number generator engine, type in the following code:
​srand((unsigned)time(0));​

Now, in order to determine a range of the random numbers that we wish to generate we have to write the following line of code which assigns an integer variable a random value determined by a given range.
​int var = 1 + rand() % (31 - 1 + 1);​
The previously written line of code will assign the ​var​ variable a randomly generated number between 1 and 31. However, the pseudo code would likely look like this ​start + rand() % (end - start + 1)​.
#include <iostream>
#include <ctime>
using namespace std;
int main(){
//RANDOM GENERATOR NUMBER ENGINE
srand((unsigned)time(0));
//ASSIGNING A RANDOM NUMBER TO VAR
//RANDOM NUMBER WILL BE BETWEEN 1 AND 31
int var = 1 + rand() % (31 - 1 + 1);
//DISPLAY RANDOMLY GENERATED NUMBER
cout << var;
return 0;
}

Custom seed

We can also provide the seed that will allow us to generate a random number based on the value that we have provided.

We start by declaring a variable of the unsigned type:
​unsigned seed;​
Then we input a value into that particular value:
​cin >> seed;​
Then we send that variable as an argument to the ​srand()​ function:
​srand(seed);​
At this point, we can now assign a random value to an interger variable; we also have to provide the ranges. The given ranges will generate a random number between 31 and 1.
​int var = 1 + rand() % (31 - 1 + 1);
#include <iostream>
#include <ctime>
using namespace std;
int main(){
//DECLARING UNSIGNED VARIABLE
unsigned seed;
//INPUT VALUE FROM USER
cin >> seed;
//USE UNSIGNED VARIABLE AS BASE TO
//CREATE RANDOM GENERATED NUMBER
srand(seed);
//DECLARE INTEGER VARIABLE TYPE
//ASSIGN RANDOM NUMBER BETWEEN 1 AND 31
int var = 1 + rand() % (31 - 1 + 1);
//DISPLAY RANDOMLY GENERATED NUMBER
cout << var;
return 0;
}

Other applications

Using this random generator technique, one could apply this method in order to expand our capabilities. For example, we could apply this random generator to a c-string variable in order to create a random c-string.

The following program uses the properties of the random number generator to create a random word generator using c-strings.
#include <iostream>
#include <ctime>
using namespace std;
int main(){
//RANDOM NUMBER GENERATOR ENGINE
srand((unsigned)time(0));
//DECLARING VARIABLE OF THE CHAR TYPE
char var[5];
//MANUALLY ASSIGNING A RANDOM LETTER TO EVERY SLOT
//OF THE VARIABLE ARRAY
var[0] = 'A' + rand() % ('z'-'A'+1);
var[1] = 'A' + rand() % ('z'-'A'+1);
var[2] = 'A' + rand() % ('z'-'A'+1);
var[3] = 'A' + rand() % ('z'-'A'+1);
var[4] = 'A' + rand() % ('z'-'A'+1);
//DISPLAY
cout << var;
return 0;
}
This program should give us a random 5 letter word every time we execute.
